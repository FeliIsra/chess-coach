# Product Agents

Colección de agentes `.md` para revisar y mejorar producto de forma sistemática.

## Agentes incluidos

1. `product-strategist.md`
   - Define dirección de producto, enfoque de valor y riesgos de posicionamiento.
2. `ux-research-analyst.md`
   - Detecta fricción de usuario, hipótesis de comportamiento y mejoras UX.
3. `growth-retention-pm.md`
   - Mejora activación, retención y loops de uso.
4. `ux-copy-content-designer.md`
   - Optimiza microcopy, claridad y tono del producto.
5. `roadmap-prioritization-pm.md`
   - Prioriza iniciativas por impacto, esfuerzo y riesgo.

## Cómo usarlos

1. Elegí un agente según el tipo de feedback que necesitás.
2. Pasale contexto concreto:
   - objetivo del producto
   - segmento de usuario
   - flujo o pantalla a revisar
   - métricas actuales (si existen)
3. Pedí salida estructurada:
   - hallazgos
   - propuestas concretas
   - prioridad
   - próximos experimentos o tareas

## Modo panel (recomendado)

Para decisiones importantes, corré 3 agentes en paralelo:

1. `product-strategist.md`
2. `ux-research-analyst.md`
3. `roadmap-prioritization-pm.md`

Después consolidá en un plan único con:

1. Qué problema resolver primero
2. Qué se implementa esta semana
3. Qué se mide para validar éxito
